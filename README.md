# Haskell Euler
My scribbles in haskell for [Project Euler](https://projecteuler.net)

## Setup & Build
Assuming you have installed the [Haskell Tool Stack](https://www.haskellstack.org), run:
```
stack setup
stack build
```

## Run All
```
stack exec haskell-euler
```

## Run Tests
```
stack test
```
